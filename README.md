-> Project Description

This project predicts the permanent magnet (PM) temperature of an electric motor using Machine Learning.

The model was trained using motor operational data such as:

Voltage

Current

Motor speed

Coolant temperature

Ambient temperature

The trained model is integrated into a Flask web application, where users can input motor parameters and get predicted temperature instantly.
